🎮 100+ Self-Hostable HTML5 Games

Open-source browser games you can host via about:blank, blob URL, or your own server

How to self-host:
1. Click a GitHub link → clone with git clone <url> or download the ZIP.
2. Open index.html directly in your browser, or serve with npx serve . or python -m http.server.
3. For blob/about:blank embedding: read the game's index.html, paste into a Blob URL using URL.createObjectURL(new Blob([html], {type:'text/html'})).
